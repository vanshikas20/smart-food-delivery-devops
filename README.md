# Smart Food Delivery System (DevOps Project)

## 📌 Description

This project is a Smart Food Delivery System developed as part of DevOps Fundamentals. It demonstrates the complete DevOps lifecycle including development, version control, CI/CD pipeline, containerization, and deployment.

## 🚀 Features

* User login and registration
* View food menu
* Place orders
* Track order status
* Admin panel for management
